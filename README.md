📖 About the Project

ZYNOVA is a modern, responsive restaurant website designed to deliver an elegant digital dining experience. The project combines beautiful typography, smooth animations, and responsive layouts to create a premium user interface suitable for restaurants, cafés, and fine dining businesses.

✨ Features
🎨 Modern and Elegant UI Design
📱 Fully Responsive on All Devices
⚡ Smooth Animations and Transitions
🍽️ Beautiful Food Showcase
📸 Gallery Section
📝 About Us Section
📋 Interactive Menu
📍 Contact & Location Information
🔥 Optimized Performance
💻 Clean and Well-Organized Code

🛠️ Technologies Used
HTML5
CSS3
JavaScript (ES6)
Google Fonts
CSS Animations
Responsive Web Design
